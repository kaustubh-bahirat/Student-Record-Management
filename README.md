Student Record Management System
Project Overview

Managing student information manually can be time-consuming and error-prone. This project implements a Student Record Management System using Java, which allows efficient storage, retrieval, and management of student data.

The system supports operations like adding, searching, updating, deleting, and displaying student records using efficient data structures.

Features
Add new student records
Display all student records
Search student by roll number
Update student details
Delete student records
Fast data retrieval using Hash Table

Data Structures Used
Hash Table (Hashing):
Enables fast lookup of student records using roll numbers.
Array:
Used to store student objects in memory.

Technologies Used
