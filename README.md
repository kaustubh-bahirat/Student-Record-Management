# Student-Record-Management

Project Description

Managing student information manually in colleges can be difficult and time-consuming. A proper system is required to store, update, search, and manage student records efficiently.

In this project, we develop a Student Management System that stores student details such as roll number, name, department, and marks. The system allows users to add, update, delete, and display student records.

This project helps us understand how data structures can be used to manage and organize data efficiently in real-world applications.

⸻

Data Structures Used

Hash Table (Hashing) – used for fast lookup of student records using roll numbers.
Array / List – used to store student information in the system.

⸻

Expected Features

Accept student details from the user
Add new student records
Display all student records
Search student by roll number using hashing
Update student details
Delete student record
Fast lookup of records using hash table
